Nobara Grub Themer
======================================
Install Bootloader Themes with ease.

Installation
---------------

Simply enter following commands in your terminal :

$ git clone https://github.com/ChrisTitusTech/Top-5-Bootloader-Themes
$ cd Top-5-Bootloader-Themes
$ sudo ./install.sh
