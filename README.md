Grub Themer
======================================
Install Bootloader Themes with ease.

Installation
---------------

Simply enter following commands in your terminal :

$ git clone https://github.com/thesacredsin/grub_themes

$ cd grub_themes

$ sudo ./install.sh

Dependencies
---------------
Requires the following dependencies:
- git
